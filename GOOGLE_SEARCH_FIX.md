## Removing Unwanted Sitelinks

Those unwanted links (калькулятор, интермигро, юр данные, архивы без рубрики) are **Google Sitelinks** - automatically generated by Google.

### Option 1: Wait for Google to Update (Recommended)
After deploying these fixes:
- Google will re-crawl your site in 1-4 weeks
- Sitelinks will update automatically based on new structure
- No action needed

### Option 2: Manual Removal via Google Search Console

1. **Go to Google Search Console:**
   - https://search.google.com/search-console

2. **Verify your site** (if not already):
   - Add property: intermigro.com
   - Verify ownership (DNS, HTML file, or Google Analytics)

3. **Request removal of unwanted sitelinks:**
   - Go to "Removals" in left sidebar
   - Click "New Request"
   - Enter the URLs you want to remove
   - Select "Temporarily remove URL from search results"

4. **Demote specific sitelinks** (Google deprecated this feature, but you can influence it):
   - Improve internal linking structure
   - Make sure unwanted pages have `noindex` meta tag
   - Remove or downgrade links to unwanted pages

### Option 3: Add noindex to unwanted pages

If those pages still exist on your site, add this to their metadata:

```typescript
export const metadata: Metadata = {
  robots: {
    index: false,
    follow: false,
  },
}
```

## What to Check After Deployment

1. **Verify robots.txt:**
   - Visit: https://intermigro.com/robots.txt
   - Should show the rules we configured

2. **Verify sitemap.xml:**
   - Visit: https://intermigro.com/sitemap.xml
   - Should only show homepage, impressum, privacy-policy

3. **Submit sitemap to Google:**
   - Go to Google Search Console
   - Sitemaps → Add new sitemap
   - Enter: https://intermigro.com/sitemap.xml
   - Click Submit

4. **Request re-indexing:**
   - In Google Search Console
   - URL Inspection → Enter your homepage URL
   - Click "Request Indexing"
